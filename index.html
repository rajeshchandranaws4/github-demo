Welcome to the Git and GitHub demonstration project

📚 What is Git?

Git is a distributed version control system that helps track changes in your code and collaborate with others.

🌐 What is GitHub?

GitHub is a cloud-based platform that hosts Git repositories and facilitates collaboration among developers worldwide.

💻 Common Git Commands

  • git init - Initialize a repository
  • git add - Stage changes
  • git commit - Save changes
  • git push - Upload to remote
  • git pull - Download from remote
